# Mission: Pythonic - Distribution Package

This package contains the compiled executable version of Mission: Pythonic.

## Installation

### Windows

1. **Extract the ZIP file** to any folder on your computer
2. **Run `MissionPythonic.exe`** to start the game
3. That's it! No Python installation required.

### Important Files

- `MissionPythonic.exe` - The game executable
- `levels/` - Level definitions (must be in the same folder as the exe)

## System Requirements

- **OS**: Windows 10 or later
- **RAM**: 512 MB minimum
- **Storage**: 50 MB free space
- **Display**: 1280x720 minimum resolution

## How to Play

1. Launch the game by double-clicking `MissionPythonic.exe`
2. Enter your agent name
3. Select a mission level
4. Write Python code to complete objectives
5. Press F5 or click RUN to execute your code

## Controls

- **Mouse**: Navigate menus and buttons
- **Keyboard**: Write code in the editor
- **F5**: Run your code
- **ESC**: Pause game

## Troubleshooting

### Game won't start
- Make sure the `levels` folder is in the same directory as the executable
- Check Windows Defender/Antivirus isn't blocking the file
- Run as Administrator if needed

### Missing dependencies error
- The executable should be self-contained
- If you see this error, re-download the package

### Save files
- Progress is saved automatically in: `%USERPROFILE%\.mission_pythonic\progress.json`
- Delete this file to reset your progress
